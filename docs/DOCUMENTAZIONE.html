1. Descrizione

Telaio3D Web è un'applicazione web per l'analisi statica lineare di telai tridimensionali. L'applicazione permette di generare geometrie parametriche, modificare tabelle di input in formato Excel, eseguire analisi strutturali con il solver OpenSeesPy e esportare i risultati.

L'app è sviluppata in Python con interfaccia Streamlit e utilizza il motore di analisi strutturale OpenSeesPy per il calcolo degli spostamenti, delle reazioni vincolari e delle sollecitazioni negli elementi.

2. Funzionalità Principali

  • Generazione parametrica di telai 3D (campate X, Y, altezze Z)
  • Editor interattivo delle tabelle input tramite UI web
  • Import/Export file Excel (.xlsx)
  • Validazione automatica del modello strutturale
  • Analisi statica lineare 3D con OpenSeesPy
  • Visualizzazione 3D della struttura e della deformata
  • Supporto per carichi concentrati nodali (Fx, Fy, Fz, Mx, My, Mz)
  • Supporto per carichi distribuiti trapezoidali lungo le aste
  • Output: spostamenti nodali e forze negli elementi

3. Input Richiesti

Il file Excel di input deve contenere i seguenti fogli:

Fogli obbligatori

Foglio Colonne Descrizione
nodes id, x, y, z Coordinate nodi (m)
elements id, n1, n2, prop, type Elementi (type = beam3d)
properties id, name, A, E, G, J, Iy, Iz Proprietà sezione e materiale
load_cases id, name Casi di carico
restraints load_case_id, node_id, ux, uy, uz, rx, ry, rz Vincoli nodali (1 = bloccato)
node_loads load_case_id, node_id, fx, fy, fz, mx, my, mz Carichi nodali

Fogli opzionali

Foglio Colonne Descrizione
dist_loads load_case_id, elem_id, qx0, qx1, qy0, qy1, qz0, qz1 Carichi distribuiti (trapezoidali)
masses load_case_id, node_id, mx, my, mz Masse nodali per analisi dinamica

4. Output Prodotti

Dopo l'esecuzione del solver, il file Excel viene arricchito con i seguenti fogli:

Foglio Colonne Descrizione
results_nodal node_id, ux, uy, uz, rx, ry, rz, Fx, Fy, Fz, Mx, My, Mz Spostamenti (m, rad) e reazioni (N, Nm)
results_elements id, n1, n2, Fx_i, Fy_i, Fz_i, Mx_i, My_i, Mz_i, Fx_j, Fy_j, Fz_j, Mx_j, My_j, Mz_j Forze locali agli estremi i e j

6. Formulario

Il solver OpenSeesPy risolve il sistema lineare:

Equazione di equilibrio
K · u = F

dove:

  • K — Matrice di rigidezza globale della struttura 3D
  • u — Vettore spostamenti nodali (6 DOF per nodo: ux, uy, uz, rx, ry, rz)
  • F — Vettore forze nodali esterne

Per elementi beam3d (asta 3D elastica):

Matrice di rigidezza locale (12×12)
A·E/L & 0 & 0 & 0 & 0 & 0 & ...
0 & 12·E·Iz/L³& 0 & 0 & 0 & 6·E·Iz/L²& ...
... & ... & G·J/L & ... & ... & ... & ...
Parametri sezione
A = Area sezione (m²)
E = Modulo di Young (MPa)
G = Modulo di taglio (MPa)
J = Costante di torsione (m⁴)
Iy = Momento d'inerzia forte (m⁴)
Iz = Momento d'inerzia debole (m⁴)

7. Esempio di Utilizzo

Schema di un telaio 3D a due campate e due piani:

Parametri di esempio:
Campate X: 4.0, 4.0 m
Campate Y: 5.0 m
Altezze Z: 3.5, 3.0, 3.0 m
Materiale: cls 30 (E = 30000 MPa)
Sezione: A = 0.30 m², Iy = 0.0225 m⁴, Iz = 0.015 m⁴
  1. Generare il telaio con i parametri nel pannello laterale
  2. Modificare le tabelle (nodes, elements, properties, ecc.)
  3. Premere Valida modello per verificare la coerenza
  4. Premere Solve ▸ Linear Static 3D
  5. Visualizzare i risultati nella tab plot3d
  6. Scaricare il file Excel con risultati
