Telaio3D Web (Streamlit + OpenSeesPy) — Come usare l’app

1) Prepara l’XLSX

Crea un file con fogli: nodes, elements, properties, load_cases, restraints, node_loads.

Opzionali: dist_loads, masses.

2) Carica e modifica

Carica l’XLSX e modifica le tabelle direttamente nella webapp.

3) Valida

Premi Valida modello per controllare colonne minime e coerenza.

4) Solve

Scegli il caso di carico e premi Solve ▸ Linear Static 3D.

5) Scarica

Scarica l’XLSX con i fogli results_nodal e results_elements.

Nota carichi trapezoidali

I carichi trapezoidali lungo l’asta vengono approssimati come somma di carichi uniformi segmentati. Aumenta lo slider “Segmenti” per maggiore accuratezza.
